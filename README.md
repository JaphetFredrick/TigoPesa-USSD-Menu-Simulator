TigoPesa USSD Menu Simulator

Description
This is a C++ console application that simulates a Tanzanian mobile money USSD menu for `TigoPesa`. It accepts a USSD code like `*150*01#`, then displays a menu with options for:
1. sending money
2. buying airtime/vouchers
3. withdrawing cash
4. paying bills
5. paying by phone
6. account services
7. financial services
8. language selection
9. offers

The program uses text input and simple validation to imitate transaction flows and user prompts.
